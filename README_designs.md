# Nely's Passion Heart Landing Page Design Versions

This folder contains the progressive design versions for the heritage café landing page.

## Files Overview

### design1.html
First full premium landing page draft.
- Built as a single-file React landing page with Tailwind CSS and Framer Motion
- Added heritage café structure: hero, story, highlights, poster, menu, cakes, hours, location, footer
- Added framed HD menu poster section
- Added separate cake cabinet section with full cake list
- Included premium Peranakan / Baba Nyonya styling direction

### design2.html
Core structure correction pass.
- Fixed menu poster image path so it loads locally
- Removed cakes from the static menu area
- Moved static menu into a sidebar layout
- Removed always-visible drink serving control
- Made hot/cold selection appear only inside drink categories

### design3.html
Menu presentation polish.
- Removed horizontal static menu navigation strip
- Kept sidebar as the primary menu navigation
- Restyled price display to feel more decorative and premium
- Added creative action buttons like "Order This" and "Save to Tea Table"

### design4.html
Simplification pass.
- Sidebar changed to title-only layout, without listing menu items inside it
- Added `cake.png` into the cake section
- Removed the action buttons from menu cards

### design5.html
Link integration pass.
- Added working Facebook link
- Added WhatsApp link
- Added café call link
- Added Google Maps link
- Added review link
- Removed Instagram button

### design6.html
Typography enhancement pass.
- Made sidebar menu titles more editorial and decorative
- Made menu item names more expressive and heritage-styled
- Improved label and tag styling in the menu section

### design7.html
Cleanup pass.
- Removed visible fallback/error-like wording
- Replaced draft-style explanatory text with cleaner landing-page language

### design8.html
More creative menu fonts.
- Changed sidebar menu title font style to a more decorative serif look
- Changed menu item names to a more elegant heritage-style serif font

### design9.html
Cursive menu font pass.
- Changed static menu typography to `Great Vibes, cursive`
- Applied to both sidebar titles and menu item names

### design10.html
Font consistency pass.
- Applied the same font style to the cake section title
- Applied the same font style to cake item names
- Cleaned another remaining static-menu sentence

### design11.html
Builder/debug-style text removal pass.
- Removed obvious explanatory sentences that sounded like internal draft notes
- Reduced landing-page copy that felt instructional instead of customer-facing

### design12.html
Additional copy cleanup.
- Removed more non-brand explanatory copy
- Further cleaned sections that still felt like design notes instead of landing-page content

### design13.html
Targeted cleanup pass.
